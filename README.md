# Netflix Clone Frontend

This project is a frontend clone of Netflix, built to replicate the user interface and experience of the popular streaming service. The project is designed to be responsive and offers various features similar to Netflix, including browsing content, watching trailers, and viewing details about movies and shows.

## Features

- **User Authentication**: Sign up, login, and logout functionality.
- **Browse Content**: View a list of movies and shows, categorized by genre.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Watch Trailers**: Preview trailers for available content.
- **View Details**: Get more information about a movie or show, including ratings and descriptions.

## Technologies Used

- **React**: For building the user interface.
- **Redux**: For state management.
- **Tailwind CSS**: For styling the components.
- **Axios**: For making API requests.
- **React Router**: For handling navigation.
- **Firebase**: For authentication and backend services.
